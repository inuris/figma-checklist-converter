# Figma Checklist Converter

A powerful Figma/FigJam widget that transforms plain text into interactive checklists with hierarchy support. Paste your notes, requirements, or meeting minutes and instantly convert them into a trackable checklist.

## Features

- **Smart Importing**: Paste multi-line text to generate tasks. Auto-detects numbered lists (e.g., "1.", "2)") as parent tasks and other lines as children.
- **Hierarchy Support**: Supports nested sub-tasks.
  - **Parent Logic**: Checking a parent task automatically checks all its sub-tasks.
  - **Child Logic**: Checking all sub-tasks automatically checks the parent task.
- **Interactive Editing**:
  - **Edit Mode**: Modify task text directly in the widget.
  - **Merge Up**: Quickly merge a task's text into the line above it.
  - **Level Switching**: Toggle any item between being a parent or a sub-task.
- **Task Management**:
  - Delete individual items.
  - Clear all completed tasks or reset the entire list.
- **Persistent State**: Your checklist state is saved with the widget instance.

## How to Use

1. **Add Tasks**: Click the blue "Add" button and paste your text.
   - Use numbers (e.g., `1. Task Name`) to create headers/parent items.
   - Regular lines underneath will become checkable sub-items.
2. **Check/Uncheck**: Click the checkboxes to track progress.
3. **Edit**: Toggle the "Edit" button to:
   - Type inside the text fields to change content.
   - Use the arrow icon on the right of a task to merge it with the previous one.
   - Click the indent icon (lines) next to the checkbox to toggle indentation level.
4. **Remove**: Toggle the "Remove" button to show delete icons (red X) next to items, or use "Clear All" to empty the list.

## Development Setup

Below are the steps to get this widget running locally for development.

### Prerequisites

- [Node.js](https://nodejs.org/en/download/) & NPM
- [Visual Studio Code](https://code.visualstudio.com/) (Recommended)

### Installation

1. Open this directory in your terminal or VS Code.
2. Install dependencies:
```bash
npm install
```

### Running the Widget
To compile the TypeScript code into JavaScript that Figma understands:

1. Watch Mode (Recommended):
Run the following command to automatically rebuild whenever you save a file:
```bash
npm run watch
```
2. One-off Build:
```bash
npm run build
```

### Project Structure
`widget-src/code.tsx`: Contains the main widget logic, UI, and state management.
`dist/code.js`: The compiled bundle generated by esbuild (do not edit manually).
For more information on Figma Widget development, visit the Widget Docs.
